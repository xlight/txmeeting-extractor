Tencent Meeting Extractor Privacy Policy

Data Collection

This extension only stores user settings and extracted meeting information locally. It does not upload any data to servers.

Data Usage

  • Uses Chrome Storage API to store user preferences
  • Extracted meeting information is saved locally only; users can export or delete it at any time

Data Security

  • All data is stored in the user's local browser
  • We do not collect, transmit, or share any user data
  • We do not use any third-party analytics tools

User Rights

Users can delete their data at any time by:

  1. Clearing data in the extension settings
  2. Clearing extension data in browser settings

Contact Information

If you have any questions, please open an issue on GitHub.
