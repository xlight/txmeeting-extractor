腾讯会议信息提取器 隐私政策

数据收集

本扩展仅在本地存储用户设置和提取的会议信息，不会上传任何数据到服务器。

数据使用

  • 使用 Chrome Storage API 存储用户偏好设置
  • 提取的会议信息仅保存在本地，用户可随时导出或删除

数据安全

  • 所有数据存储在用户本地浏览器中
  • 不收集、不传输、不共享任何用户数据
  • 不使用任何第三方分析工具

用户权利

用户可随时通过以下方式删除数据：

  1. 在扩展设置中清除数据
  2. 在浏览器设置中清除扩展数据
