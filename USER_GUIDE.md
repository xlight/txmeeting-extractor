# 使用说明

## 安装

### 从源码构建

1. **克隆或下载项目**

   ```bash
   git clone <repository-url>
   cd txmeeting-extractor
   ```

2. **安装依赖**

   ```bash
   pnpm install
   ```

3. **构建扩展**

   ```bash
   pnpm run build
   ```

   构建完成后，扩展文件将生成在`dist/`目录中。

4. **加载到Chrome**
   - 打开Chrome浏览器
   - 访问 `chrome://extensions/`
   - 开启右上角的"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择项目的`dist`文件夹

## 使用方法

### 第一步：访问腾讯会议云录屏页面

1. 打开腾讯会议的云录屏页面
   - URL格式通常为：`https://meeting.tencent.com/cw/xxxxx`
2. 等待页面加载完成
3. 扩展会自动在后台拦截并缓存会议数据

### 第二步：查看会议信息

1. 点击Chrome工具栏上的扩展图标（蓝色TM图标）
2. 弹出窗口将显示提取的会议信息，包括：
   - 📊 会议信息（ID、时长、时间等）
   - 💡 智能总结
   - 📝 会议纪要
   - 🎙️ 转写内容（完整对话记录）
   - 👥 参会人员
   - 💬 聊天记录
   - ✅ 行动项
   - ⭐ 重点时刻

### 第三步：导出Markdown

1. 在弹出窗口中，点击"📥 导出 Markdown"按钮
2. 浏览器将自动下载一个Markdown文件
   - 文件名格式：`会议标题-YYYY-MM-DD.md`
3. 使用任何Markdown编辑器打开文件查看内容

## 功能特性

### ✨ 自动提取

- 无需手动操作，扩展自动拦截API响应
- 支持实时更新（刷新页面时重新抓取最新数据）
- 本地缓存，无需重复加载

### 📋 便捷复制

- 每个数据section都有独立的"复制"按钮
- 一键复制转写内容、总结、纪要等
- 支持快速分享到聊天工具或笔记应用

### 📁 灵活展示

- 可折叠的分节展示，节省空间
- 长内容自动滚动
- 清晰的层级结构

### 🔄 数据刷新

- 点击"🔄 刷新"按钮重新加载会议数据
- 确保获取最新的会议信息

## 常见问题

### ❓ 弹出窗口显示"未找到会议数据"

**可能原因**:

- 您还未访问过腾讯会议云录屏页面
- 页面加载未完成，API请求尚未触发

**解决方法**:

1. 确保访问的是腾讯会议云录屏页面（URL包含`meeting.tencent.com/cw/`）
2. 等待页面完全加载
3. 刷新页面重新触发数据加载
4. 点击扩展的"刷新"按钮

### ❓ 部分数据字段为空

**可能原因**:

- 会议未启用某些功能（如智能总结、纪要等）
- API响应中不包含这些数据

**说明**:
这是正常现象。扩展只显示API返回的数据，如果某个功能未在会议中使用，对应section将不会显示或显示为空。

### ❓ 转写内容时间戳不准确

**说明**:
转写时间戳来自腾讯会议API，扩展不做修改。如有不准确，属于腾讯会议服务本身的问题。

### ❓ 导出的Markdown文件中文乱码

**解决方法**:
确保使用支持UTF-8编码的Markdown编辑器打开文件。推荐工具：

- VS Code
- Typora
- Obsidian
- Notion（导入功能）

## 隐私说明

### 🔒 数据安全

- ✅ 所有数据仅存储在您的浏览器本地
- ✅ 不上传任何数据到外部服务器
- ✅ 不收集任何个人信息
- ✅ 仅在腾讯会议页面激活

### 🗑️ 清除数据

如需清除缓存的会议数据：

1. 方法一：右键点击扩展图标 → "选项" → 清除缓存（如已实现）
2. 方法二：访问 `chrome://extensions/` → 找到扩展 → 点击"详细信息" → "删除扩展数据"
